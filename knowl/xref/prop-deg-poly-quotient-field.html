Proposition 2.5. Degree of \(F[x]/(p(x))\).

Let \(F\) be a field and assume \(p(x) \in F[x]\text{.}\) Then, \([F[x]/(p(x)) : F] = \deg p\text{.}\)
🔗
🔗

Proof.

Put \(L = F[x]/(p(x))\text{,}\) let \(\theta = \pi(x)\) for the quotient map \(\pi: F[x] \rightarrow L\text{,}\) and assume \(n = \deg p\text{.}\) We will show that \(L\) has basis \(B = \{1, \theta, \theta^2, \ldots, \theta^{n - 1}\}\text{.}\)
  1. \(B\) is linearly independent. Assume there exist coefficients \(a_{n - 1}, \ldots, a_0\text{,}\) such that \(a_{n - 1}\theta^{n - 1} + \cdots + a_0 = 0\text{.}\) Let \(q(x) = a_{n - 1}x^{n - 1} + \cdots + a_0\text{.}\) It must be true that \(\pi(q(x)) = 0\text{.}\) Then, \(q(x) \in (p(x))\text{.}\) However, \(\deg q < n\) and \(\deg p = n\text{.}\) The only \(q\) satisfying these properties is \(q(x) = 0\) so all \(a_0, \ldots, a_{n - 1}\) must be 0, as required.
    🔗
  2. \(B\) spans \(L\text{.}\) \(F[x]\) is spanned by \(\{1, x, x^2, \ldots\}\) so \(L\) is spanned by \(\{1, \theta, \theta^2, \ldots\}\text{.}\) Now, assume without loss of generality that \(p\) is monic and \(p(x) = x^2 + a_{n - 1}x^{n - 1} + \cdots + a_0\text{.}\) We have
    \begin{equation*} \theta^n = -a_{n - 1}\theta^{n - 1} - \cdots - a_0 \end{equation*}
    since \(\pi(p(x)) = 0\text{.}\) Consider \(\theta^{n + 1}\text{.}\) It follows
    \begin{equation*} \theta^{n + 1} = -a_{n - 1}\theta^n - \cdots - a_0\theta. \end{equation*}
    It follows, by induction,
    \begin{equation*} \theta^{n + k} = -a_{n - 1}\theta^{n + k} - \cdots - a_0\theta^k\text{.} \end{equation*}
    Therefore, every \(\theta^n, \theta^{n + 1}, \ldots\) is a linear combination in the span of \(B\) and
    \begin{equation*} \langle B\rangle = \langle 1, \ldots, \theta^n\rangle = \langle 1, \ldots, \theta^{n + 1}\rangle = \langle 1, \theta, \theta^2, \ldots\rangle = L. \end{equation*}
    🔗
🔗
Consequently, \(B\) is a basis of \(L\) so \([L:F] = n\text{,}\) as required.
