    Section 3 4-3-26

    Field Extension Isomorphisms, Field of Rational Functions, Tower Law.
    🔗
    Notes courtesy of Jack Thilin.
    🔗

    Definition 3.1. Isomorphism of Field Extensions.

    Given \(L_1/F\) and \(L_2/F\text{,}\) say \(L_1\) and \(L_2\) are isomorphic if there exists a ring isomorphism \(\varphi: L_1 \overset{\scriptstyle{\sim}}{\rightarrow} L_2\) making the diagram
    commute.
    🔗
    🔗

    Corollary 3.2. Algebraic Elements and Extension Isomorphisms.

    Given \(L/F\text{,}\) irreducible \(p \in F[x]\text{,}\) and roots \(\alpha, \beta \in L\) of \(p\)
    \begin{equation*} F(\alpha) \cong F(\beta) \end{equation*}
    as extensions of \(F\text{.}\)
    🔗
    🔗

    Proof.

    From Corollary 2.9, we have \(F(\alpha) \cong F[x]/(p(x)) \cong F(\beta)\text{.}\)
    🔗
    🔗

    Example 3.3.

    Let \(\xi = \frac{-1 + \sqrt{-3}}{2}\text{.}\) Then,
    \begin{equation*} x^3 - 2 = (x - \sqrt[3]{2})(x^2 + \sqrt[3]{2}x + \sqrt[3]{4}) = (x - \sqrt[3]{2})(x - \xi \sqrt[3]{2})^2\text{.} \end{equation*}
    From Corollary 3.2, \(\Q(\sqrt[3]{2}) \cong \Q(\xi \sqrt[3]{2})\text{.}\)
    🔗
    🔗

    Definition 3.4. \(\textrm{Frac}(F[x])\).

    Define \(\textrm{Frac}\) such that \(\textrm{Frac}(F[x]) = F(x)\text{.}\)
    🔗
    Note that \(F(x)\) is the smallest field containing both the indeterminate \(x\) and \(F\text{.}\) \(F(x)\) is in fact the field of rational functions.
    🔗
    🔗

    Proposition 3.5. Degree of \(\textrm{Frac}(F[x])\).

    \([F(x):F] = \infty\text{.}\)
    🔗
    🔗

    Proof.

    \(F[x] \subseteq F(x)\) has basis \(\{1, x, x^2, \ldots\}\) over \(F\text{.}\) By the Tower Law,
    \begin{equation*} [F(x):F] = [F(x):F[x]] \cdot [F[x]:F] = [F(x):F[x]] \cdot \infty = \infty. \end{equation*}
    🔗
    🔗

    Proposition 3.6. Transcendental Elements and Extension Isomorphisms.

    Let \(L/F\) and \(\alpha \in L\text{.}\) If \(\alpha\) is transcendental, then the extensions \(F(x)\) and \(F(\alpha)\) are isomorphic.
    🔗
    🔗

    Proof.

    Let \(\phi: F[x] \rightarrow F(\alpha)\) be the evaluation map at \(\alpha\text{.}\) Since \(\alpha\) is transcendental, there exists no \(p \in F[x]\backslash \{0\}\) such that \(p(\alpha) = 0\text{.}\) Consequently, \(\ker \phi = \{0\}\text{,}\) so \(\phi\) is injective.
    🔗
    Then,
    and as the inclusion map from \(F[x]\) to \(F(x)\) is injective and \(\phi\) is injective, it follows that \(\varphi\) is injective. Furthermore, \(\textrm{im } \varphi\) contains \(F\) and \(\alpha\text{,}\) so \(\varphi\) is an isomorphism.
    🔗
    🔗

    Definition 3.7. Algebraic Extensions.

    An extension \(L/F\) is algebraic if every \(\alpha \in L\) is algebraic over \(F\text{.}\)
    🔗
    🔗

    Proposition 3.8. Degree and Algebraic Extensions.

    If \([L:F] < \infty\text{,}\) then \(L/F\) is algebraic.
    🔗
    🔗

    Proof.

    Assume \(L/F\) is not algebraic. Then, there exists transcendental \(\alpha \in L/F\text{.}\) From Proposition 3.6, we have \(F(\alpha) \cong F(x)\text{.}\) From Proposition 3.5, it follows that \([F(\alpha):F] = \infty\text{.}\) As \(F(\alpha)\) is intermediate to \(L/F\text{,}\) by the Tower Law, we have the contradiction \([L:F] = \infty\text{.}\)
    🔗
    🔗

    Theorem 3.9. Tower Law.

    Suppose \(K/L/F\text{.}\) Then,
    \begin{equation*} [K:F] = [K:L][L:F]. \end{equation*}
    🔗
    🔗

    Proof.

    If either \([K:L] = \infty\) or \([L:F] = \infty\text{,}\) it is easy to see that \([K:F] = \infty\text{.}\)
    🔗
    Otherwise, let \([L:F] = d\) and \([K:L] = e\text{.}\) Let \(U = \{u_1, \ldots, u_d\}\) be a basis for \(L/F\) and \(V = \{v_1, \ldots, v_e\}\) be a basis for \(K/L\text{.}\)
    🔗
    We will now show that \(B = \{u_i v_j \mid 1 \le i \le d, 1 \le j \le e\}\) is a basis for \(K/F\text{.}\)
    1. \(B\) spans \(K/F\text{.}\) Suppose \(v \in K\text{.}\) Since \(V\) is a basis for \(K/L\text{,}\) there exist \(c_1, \ldots, c_e \in L\) such that \(v = \sum_{i = 1}^{e} c_iv_i\text{.}\) Furthermore, since each \(c_i \in L\) and \(U\) is a basis for \(L/F\text{,}\) there exists \(a_{i1}, \ldots, a_{id} \in F\) such that \(\sum_{j = 1}^{d} a_{ij}u_j = c_i\text{.}\) It follows that
      \begin{equation*} v = \sum_{i = 1}^{e} \left(\sum_{j = 1}^{d} a_{ij} u_j\right) v_j = \sum_{i = 1}^{e} \sum_{j = 1}^{d} a_{ij} (u_jv_i) \end{equation*}
      and \(v\) is a linear combination of \(B\text{.}\)
      🔗
    2. Assume
      \begin{equation*} 0 = \sum_{i = 1}^{e} \sum_{j = 1}^{d} a_{ij}(u_jv_i) = \sum_{i = 1}^{e} \left(\sum_{j = 1}^{d} a_{ij}u_j\right)v_i. \end{equation*}
      Since \(V\) is linearly independent, it must be the case that each \(\sum_{j = 1}^{d} a_{ij}u_j = 0\text{.}\) Furthermore, as \(U\) is linearly independent, this only occurs if each \(a_{ij} = 0\text{,}\) as required.
      🔗
    🔗
    Consequently, \(B\) is a basis of \(K/F\) so \([K:F] = ed = [K:L][L:F]\text{.}\)
    🔗
    🔗

    Example 3.10.

    Assume \(\textrm{char } F \ne 2\) and \(L/F\) has degree 2. This suggests \(L = F(\alpha)\) for some \(\alpha \in L/F\) with a minimal polynomial \(m_\alpha(x) = x^2 + bx + c\) for \(b, c \in F\text{.}\) Note that \(F(\alpha)\) has degree 2 from Corollary 2.9 and Proposition 2.5.
    🔗
    Additionally, \(\alpha + \frac{b}{2}\) has minimal polynomial
    \begin{equation*} (x - \frac{b}{2})^2 + b(x - \frac{b}{2} + c = x^2 - \frac{b^2}{4} + c\text{.} \end{equation*}
    Consequently, \(L \cong F \left(\sqrt{\frac{b^2}{4} - c}\right)\text{.}\)
